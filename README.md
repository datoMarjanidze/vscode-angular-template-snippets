## VS Code Angular Template Snippets

This extension contains template snippets that can be used in HTML files.
Angular version: 11.2.4

* NgComponentOutlet

* NgComponentOutletNgModuleFactory

* NgIf
* NgPlural
* NgPluralCase
* NgStyle
* NgSwitch
* NgSwitchCase

* NgTemplateOutlet
